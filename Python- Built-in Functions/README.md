# Python Built-in Functions

공식문서를 찾아보면 사용 예시나 설명문이 빈약해서 추가로 블로그를 여러개 찾아본 경험이 있으실 겁니다.

그래서 공식문서와 함께 쓸만한 예시를 작성한 글입니다.

메서드를 소개하는 문서마다, 파이썬 공식 메서드에 대한 설명의 번역과 원문을 첨부해놨습니다.

직접 번역한 내용과 함께 개인적인 설명과 예시를 추가했습니다.
